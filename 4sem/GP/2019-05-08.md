## Tipos de Reuniões
Há classicamente 3 tipos
- Analise de Status
	- Situação do Projeto
- Resolução de Problema
	- Resolver conflitos ou problemas
- Analise Critica de Concepção Técnica
	- Reunião de levantamento de requisitos
