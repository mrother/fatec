# Funções de Data

`CURRENT_DATE()` - Retorna a data atual
`CURRENT_TIME()` - Retorna a hora atual
`CURRENT_TIMESTAMP()` / `NOW()` - Retorna o timestamp atual
`DAYNAME()`
`DAYOFWEEK()`
`DAYOFMONTH()`
`DAYOFYEAR()`

# Outras Funções
`CASE`

`COALESCE` - Retorna, da esq. para dir. o primeiro parâmetro não-nulo
`IF`
