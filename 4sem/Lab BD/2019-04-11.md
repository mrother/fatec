# Funções numéricas
### DIV
Divisão inteira
```sql
SELECT 5 DIV 2
```

## MOD
Resto da divisão inteira
```sql
SELECT 5 MOD 2
```

### RAND
Randômico
```sql
SELECT ator_id, primeiro_nome
FROM ator
ORDER BY RAND()
```
